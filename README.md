# Vim Configurations

My .vim dotfiles and configurations. 	

# Install

Just run the following commands via terminal to get perfectly set up:

    $ cd ~/    
    $ git clone https://github.com/luizpicolo/vim.git .vim && ln -sf ~/vim/.vimrc ~/.vimrc && ln -sf ~/vim/.vim ~/.vim
